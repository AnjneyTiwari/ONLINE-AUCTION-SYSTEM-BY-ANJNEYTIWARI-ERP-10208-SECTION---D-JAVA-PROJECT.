# Online Auction System

## Description
A simple console-based Online Auction System developed in Java using Object-Oriented Programming concepts.

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Arrays
- String Handling
- Exception Handling
- Access Modifiers
- Method Calls

## Files
1. Main.java
2. Item.java
3. Bidder.java

## Features
- View auction items
- Place bids
- Validate bid amount
- View bidder details
- Exception handling for invalid input

## Sample Items
- Laptop
- Smart Phone
- Gaming Headset

## Developed Using
Java
